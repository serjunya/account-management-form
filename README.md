# Account Management Form

Небольшое приложение на Vue 3 + Vite для ведения учетных записей. Оно позволяет добавлять записи с метками, выбирать тип учетной записи (LDAP или локальная), указывать логин и пароль, а также удалять строки.

## Возможности

- Добавление и удаление учетных записей.
- Метки для учетной записи вводятся через разделитель `;`.
- Поддержка типов учетных записей: LDAP и LOCAL.
- Валидация полей на длину и обязательность (логин/пароль для локальных учетных записей).

## Требования

- Node.js 18+ (рекомендуется).
- npm 9+.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

По умолчанию приложение доступно на `http://localhost:4173`.

## Сборка

```bash
npm run build
```

## Предпросмотр сборки

```bash
npm run preview
```

## Проверка типов

```bash
npm run typecheck
```

## Структура проекта

- `src/App.vue` — основной экран со списком учетных записей.
- `src/components/AccountRow.vue` — строка формы для редактирования записи.
- `src/stores/accounts.ts` — хранилище Pinia с логикой управления данными.
- `src/styles.css` — базовые стили приложения.

## Заметки

- Пароль обязателен только для локальных учетных записей.
- Для LDAP пароль не сохраняется.
